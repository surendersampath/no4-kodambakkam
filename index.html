The columns are extracted from the drawing's vector geometry (the magenta column-marking rectangles only) and framed into a stilt + 3-storey skeleton.

From the drawing

  • 20 columns, each on its grid node to within ~0.01 ft. Sections 9"×1'-6", 9"×2' and rotated 1'-6"×9".
  • Staggered layout — row 1 is A·C·D (no B), row 3 is D only, etc. Click any column to read its size.
  • The blue line is the site boundary (as per survey), traced from the drawing.

Frame (stilt + 3, 10'-0" storeys)

  • Stilt + 3 storeys at 10'-0" floor-to-floor (≈40'-0" to roof), with tie/floor beams between columns at every level and a roof slab. The stilt is left open.
  • North is taken from the drawing's title-block arrow, which points toward grid A: North = grid A side, South = grid D, East = row 1, West = row 12. A ground compass and cardinal labels show this.
  • Beams follow each grid line through the columns on it — an indicative skeleton, not a stamped framing plan.
